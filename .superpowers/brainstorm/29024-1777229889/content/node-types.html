What node types can users add?

Backend executor already supports: model, passthrough, prompt_template. Which should be exposed in UI?

A

Model node

Pick any of the 7 white-label models (Apex, Swift, Prism…). Gets input from upstream, streams a response. Core node.

B

Prompt template node

Inject a static system prompt or transformation before a model. E.g. "Translate the above to Spanish, then pass to Swift."

C

Scout (web search) node

Runs a web search, injects results as context into the next node.

D

Merge node

Combines outputs of multiple upstream nodes into one input. Useful for fan-out → synthesize patterns.
