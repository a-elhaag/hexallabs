How do users wire nodes together?

How connections are created between nodes on the canvas.

A

Drag port → port

Each node has an output port (right) and input port (left). Drag from output to input to connect. Standard node-graph UX (like Figma, n8n). Most flexible.

B

Click to connect

Click a node's output → it highlights → click another node's input to connect. No drag needed. Better for touch/trackpad.

C

Auto-chain (linear only)

Nodes snap into a left→right chain automatically. Add node → it attaches to the last one. No manual wiring. Simple but no branching/merging.
