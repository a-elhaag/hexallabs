Where does the Workflow builder live?

When user selects Workflow mode, where does the node canvas appear?

A

Slide-up panel above chat input

Canvas expands upward from the input area, replacing the message list. Chat input stays pinned at bottom. Compact — no page navigation.

B

Full-width overlay / modal

Clicking Workflow opens a full-screen canvas over the chat. Build the pipeline, click Run — canvas closes and streams results into chat.

C

Side-by-side split

Canvas occupies left ~55%, chat/results on right ~45%. Both visible at once. Wider screen required.
